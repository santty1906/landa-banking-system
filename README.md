# LandaProject

Monorepo scaffold for a mobile banking Software Quality project.

## Structure

- `frontend/` Flutter mobile app (scaffold only)
- `backend/` FastAPI backend (scaffold only)
- `docs/` project documentation
- `.github/workflows/` CI templates

## Quick start

1. Clone repository
2. Copy environment examples
3. Start local services with Docker
4. Run backend and frontend locally

See `docs/setup.md` for full setup steps.

## Status

Project setup and architecture foundations only. Business logic is intentionally not implemented yet.
